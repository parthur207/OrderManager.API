## Funcionalidade por papel de usuário

### Administrador - Querys

* Consulta de todos os usuários existentes 
* Consulta de usuário por e-mail
* Consulta de usuário por número do pedido

* Consulta de todos os pedidos já realizados
* Consulta de todos os pedidos por email do usuário solicitante
* Consulta do pedido pela seu número
* Consulta de todos os pedidos pelo tipo de ocorrencia presente

### Administrador - Comandos

* Login
* Inativa um usuário pelo email
* Ativa um usuário pelo email
* Deleta um usuário pelo email
* Promove um usuário para "Administrador" (Adm)

* Cria uma nova ocorrencia a um pedido
* Deleta uma ocorrencia de um pedido

### Usuário Comum - Querys
* Consulta todos os pedidos atrelados a seu ID

### Usuário comum - Comandos
* Login
* Cadastro
* Cria um novo pedido
